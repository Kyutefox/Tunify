# Contributing

Thank you for your interest in contributing to Tunify.

## Setup

1. Install [Flutter](https://docs.flutter.dev/get-started/install) **^3.6.0**.
2. Clone the repo and run `flutter pub get`.
3. Run `dart run build_runner build --delete-conflicting-outputs` when you change Freezed models.

## Guidelines

- Run `flutter analyze` before opening a PR.
- Follow existing style; `flutter_lints` is enforced.
- Prefer focused PRs (one feature or fix).
- Do not commit secrets or API keys.
